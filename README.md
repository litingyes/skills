# Skills Project Overview

This repository maintains reusable skills for engineering workflows and skill governance.

## Current Root Skills

Scope: only non-hidden skill directories under the repository root (directories containing `SKILL.md`).

| Skill | Summary | Why It Exists |
| --- | --- | --- |
| `development-frontend` | Defines frontend engineering baselines, code quality checks, commit/release workflows, and legacy project audit guidance. | Provides a consistent frontend lifecycle standard to reduce quality variance and collaboration friction. |
| `skills-constitution` | Defines governance for creating, updating, auditing, and evaluating skills. | Keeps the skill lifecycle controllable and prevents trigger drift, structural inconsistency, and quality regression. |
